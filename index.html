Socket Calculator Program

Socket Calculator Program is a Python program that establishes a basic client-server interaction using sockets for arithmetic calculations. The client connects to the server, sends mathematical expressions, and receives results. The server, upon receiving expressions, performs calculations and responds with the results. The communication continues until either the client inputs '0/0' to end its program or the server receives '0/0=' and concludes. This simple yet functional system allows users to input arithmetic expressions via the client and receive computed results from the server, making it a concise and illustrative example for basic client-server communication in Python.

Database Management Application

This project involves developing a Java-based GUI application that connects to a MySQL server via JDBC. The application supports client-side execution of SQL commands against various databases, allowing for both general and specialized use cases. Users can issue SQL DDL or DML commands, with results displayed through the GUI. Additionally, the application features a monitoring interface for an accountant client to view transaction logs. The system maintains separate logging for queries and updates, handling multiple users with varying permissions.
